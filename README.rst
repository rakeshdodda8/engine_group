Engine Group Puzzle
===================

Coding challenge for implementing resume submmision using Python `Flask` framework.

Install
-------
    You need to have Python 2.x / 3.x and pip set-up on your machine.

    PIP
    ===
    pip is the package installer for Python. You can use pip to install packages from the Python Package Index and other indexes.

Build Instructions
------------

    # clone the repository
    $ git clone git@github.com:rakeshdodda8/engine_group.git
    $ cd engine_group

Create a virtual environment and activate it::
    
    $ pip install virtualenv
    $ virtualenv env
    $ source env/bin/activate

Install Application::

    $ pip install -e .


Run
---

::

    $ export FLASK_APP=puzzle
    $ flask run
